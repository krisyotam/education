# Education

A self-directed curriculum scaling knowledge to PhD-level and beyond across multiple disciplines.

## Knowledge Framework

All resources are organized around three types of knowledge:

- **Declarative** — Facts, definitions, theorems, concepts. Learned through spaced repetition (Anki, etc.)
- **Procedural** — Skills, problem-solving, proofs, coding, lab work. Learned by doing.
- **Theory** — Deep understanding, connections between ideas, intuition. Built through reading, lectures, and reflection.

## Subjects

- `math/` — Pure and applied mathematics
- `cs/` — Computer science
- `literature/` — Literature and literary theory
- `marine-biology/` — Marine biology and oceanography
- `philosophy/` — Philosophy
- `psychology/` — Psychology

## Structure

Each subject contains:
- `sources/` — Curated resources: textbooks, lecture series, courses, reading lists, problem sets
- `reports/` — Notes, summaries, progress tracking
