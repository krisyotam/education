## Data Structures and Algorithms

### Declarative
- Time/space complexity for all standard data structures (arrays, linked lists, stacks, queues, hash tables, BSTs, heaps, graphs)
- Sorting algorithms and their complexities (quicksort, mergesort, heapsort, radix sort)

## Operating Systems

### Declarative
- Process management: states, scheduling algorithms (FCFS, SJF, round-robin, MLFQ, CFS)
- Threads, concurrency primitives (mutexes, semaphores, condition variables, monitors)
- Memory management: paging, segmentation, page replacement (LRU, clock, optimal)

## Distributed Systems

### Declarative
- CAP theorem, PACELC, consistency models (linearizability, eventual, causal)
- Consensus: Paxos, Raft, Byzantine fault tolerance

## Complexity Theory

### Declarative
- P, NP, NP-complete, NP-hard, co-NP
- Cook-Levin theorem, common NP-complete problems (SAT, 3-SAT, clique, vertex cover, Hamiltonian cycle, TSP)
- Polynomial hierarchy, PSPACE, EXPTIME
- Randomized complexity: BPP, RP, ZPP
- Circuit complexity, communication complexity
- Approximation algorithms and hardness of approximation (PCP theorem)
- Interactive proofs, zero-knowledge proofs (IP = PSPACE)
